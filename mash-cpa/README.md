# Henderson Financial - CPA Website

Professional website for Henderson Financial CPA services built with React, TypeScript, and Vite.

## Prerequisites

- Node.js (v18 or higher recommended)
- npm, yarn, or pnpm package manager

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run linter
npm run lint
```

## Project Structure

- `src/pages/` - Page components (Home, Contact, Services)
- `src/components/` - Reusable components (Navbar, Footer, Forms)
- `src/pages/services/` - Service-specific pages (Tax Planning, Bookkeeping, etc.)
- `public/` - Static assets

## Key Features

- Multiple service pages with detailed information
- Lead generation forms integrated with LeadConnector
- Google reviews integration
- Responsive design with Tailwind CSS
- Client-side routing with React Router

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Lucide React (icons)
- React Router DOM

## Deployment

Production builds are generated in the `dist/` directory. Suitable deployment platforms include:
- Vercel (recommended)
- Netlify
- GitHub Pages
